Allgemeine Fehlermeldungen

Status Name Botschaft Ursache
400 BAD_REQUEST Invalid JSON body on request In JSON ist ein Fehler für den Anforderungshauptteil aufgetreten. Dies kann ein grundlegendes JSON-Problem oder ein falscher Typ für einen Feldwert sein
400 BAD_REQUEST "error": "{fieldname} field is required Dem Anforderungshauptteil fehlt das angegebene erforderliche Feld
400 AD_CONFIG_NOT_FOUND Ad config not found Die in einer SSAI-Anforderung angegebene Anzeigenkonfiguration wurde nicht gefunden
400 AD_CONFIG_INACTIVE Ad config is inactive Die in einer SSAI-Anforderung angegebene Anzeigenkonfiguration ist inaktiv
401 UNAUTHORIZED Permission denied. Fehlendes oder ungültiges OAuth-Zugriffstoken. Wenn Sie ein Zugriffstoken angegeben haben, verfügt es möglicherweise nicht über den entsprechenden Umfang für diese Anforderung
403 NOT_AVAILABLE Resource is not available. Die Ressource, die Sie anfordern, ist vorübergehend nicht verfügbar. Dies kann eine vorübergehende Bedingung sein, während eine Verarbeitung des Videos ausgeführt wird. Wenn die Nachricht jedoch weiterhin angezeigt wird, wenden Sie sich an den Support.
404 NOT_FOUND Resource does not exist. Sie haben eine Ressource angefordert, die nicht vorhanden ist. Möglicherweise versuchen Sie, eine Anzeigenkonfiguration zu aktualisieren oder eine Anzeigenablaufverfolgung für eine Sitzung abzurufen, die nicht mehr vorhanden ist. Dieser Fehler wird auch zurückgegeben, wenn Sie eine HTTP-Methode verwenden, die auf dem Endpunkt nicht zulässig ist.
500 UNKNOWN an unknown internal error occurred Problem im Brightcove-System - versuchen Sie es später erneut.
503 SERVICE_UNAVAILABLE The API is temporarily unavailable Backend-Problem - versuchen Sie es später erneut.
504 TIMEOUT something took too long Server wahrscheinlich zu beschäftigt - versuchen Sie es später erneut.
